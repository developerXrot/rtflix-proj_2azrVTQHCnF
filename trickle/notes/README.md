# Rtflix Xiters - منصة إدارة السكريبتات

## نظرة عامة
Rtflix Xiters هو موقع مخصص لعرض وإدارة السكريبتات والأدوات الرقمية بشكل منظم واحترافي، مع لوحة تحكم داخلية لتسهيل إدارة المحتوى والإحصائيات.

---

## الصفحات

- `index.html`  
  الصفحة الرئيسية لعرض السكريبتات، الأدوات، وآخر الإضافات بشكل احترافي للمستخدمين.

- `admin1891.html`  
  لوحة التحكم الخاصة بالإدارة، تُستخدم لإضافة وتعديل وحذف السكريبتات ومتابعة الأداء.

---

## الميزات

- عرض السكريبتات داخل واجهة حديثة ومنظمة
- صورة مصغرة (Thumbnail) لكل سكريبت
- وصف مختصر لكل أداة
- روابط متعددة للتحميل
- لوحة تحكم داخلية للإدارة
- إضافة / تعديل / حذف السكريبتات
- تتبع عدد المشاهدات والتحميلات
- إحصائيات الاستخدام والزوار
- تصميم سريع ومتجاوب مع الهاتف والحاسوب

---

## قاعدة البيانات

### `script`
جدول خاص بتخزين بيانات السكريبتات، ويحتوي على:

- اسم السكريبت
- الوصف
- الصورة المصغرة
- روابط التحميل
- تاريخ الإضافة
- عدد المشاهدات
- عدد التحميلات

### `stats`
جدول خاص بتخزين الإحصائيات العامة، مثل:

- عدد الزوار
- عدد التحميلات
- أكثر السكريبتات مشاهدة
- أكثر السكريبتات تحميلاً

---

## لوحة التحكم

لوحة الإدارة (`admin1891.html`) تتيح:

- إضافة سكريبت جديد
- تعديل بيانات سكريبت موجود
- حذف سكريبت
- تحديث روابط التحميل
- مراقبة الإحصائيات
- إدارة المحتوى بسرعة

---
