# Trickle Rules - README.md Update Policy

## الهدف
هذا الملف يحدد القواعد الرسمية لتحديث وتعديل ملفات `README.md` داخل مشروع Rtflix Xiters / Trickle، لضمان توثيق منظم ودقيق يعكس حالة المشروع دائماً.

---

## 1. التحقق قبل التحديث

عند أي تعديل في المشروع يجب:

- فحص إذا تمت إضافة ميزات جديدة
- تحديد إذا كانت تحتاج توثيق داخل README.md
- التأكد من أن التغييرات تؤثر على:
  - الصفحات
  - الوظائف
  - قاعدة البيانات
  - لوحة التحكم

---

## 2. نظام الإصدارات (Version System)

### صيغة الإصدارات:
- `v1.0` → النسخة الأساسية الأولى
- `v1.1` → تحسينات أو ميزات صغيرة
- `v1.2` → تحسينات إضافية
- `v2.0` → تغيير كبير في النظام أو التصميم

### قواعد الترقيم:
- إضافة ميزة جديدة → رفع رقم فرعي
- تغيير كبير → رفع الرقم الرئيسي
- إصلاحات بسيطة → بدون تغيير كبير إلا إذا كان مهم

---
